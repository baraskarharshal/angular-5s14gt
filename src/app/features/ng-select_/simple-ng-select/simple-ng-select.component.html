Key Points

  • Default value of formcontrol should be null to make placeholder visble.
  • Use appendTo="body" when dropdown is hiding behind.
  • User bindValue="propertyName" to get value of specific property from selected object. If bindValue is not used then by default complete object is selected.
  • Autocomplete is on by default.
  • Use multiple="true" to allow multiple selection.
